How It Works

Step 1 Average RGB

Each cell subsamples its area of the source image and averages the red, green, and blue channels into a single color.

Step 2 Luminance

That RGB color is converted to a brightness value using standard luminance weights. (Here's the math, if you're curious and/or you like math.)

Step 3 Tone Bucket

Those brightness values are divided into one of 10 tone "buckets," determining which tile or glyph to draw.

Step 4 Tile Render

Each final tile is drawn based on the user settings with the average color as a background and the tile overlaid.

About this tool

I created this tool as part of a curriculum on the Elements and Principles of Design. It is intended to help design students explore the Elements of Line, Texture, and Value, as well as the Principles of Contrast, Pattern (Repetition / Rhythm), and Scale / Proportion.

This tool creates mosaics in the opus regulatum style, in which tiles or tessarae are placed according to a regular grid. Example pieces created with this tool can be found here. Tag or message me if you find this tool useful—I'd love to see what you make with it!
